Geister / ガイスター

What's this

A German 2 person board game using ghost-based piecs.
Geister(ガイスター)はおばけのコマを用いるドイツ製の二人用ボードゲームです．
これはガイスターをWEB上でできるようにしたものです．

Victory Conditions

  1. Take all the opponent's good ghost. / 相手の良いおばけをすべて取る
  2. Take everything against your bad ghost. / 自分の悪いおばけをすべてとらせる
  3. Escape one's own good ghost away from corner of opponent side. / 自分の良いおばけを相手側の角のマスから逃がす

How To Play

  1. Player 0 Create a new game
  2. Player 1 Enter the room id that is displayed on Player0's window.
  3. Click on join game.
